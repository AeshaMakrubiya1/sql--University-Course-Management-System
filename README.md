# 📚 Multi-Stack Development Portfolio

A comprehensive collection of projects demonstrating proficiency across **Python (Functional & OOP)**, **Data Analysis**, and **Relational Database Management (SQL)**.

---

## 🛠️ Project Modules

### 1. 🧪 Functional Treat (Python)

A deep dive into Python's functional programming capabilities.

* **Core Concepts:** Implements recursion for factorials, lambda functions for data filtering, and higher-order functions like `map()` and `filter()`.
* **Features:** Supports both 1D list processing and 2D matrix manipulation.
* **Technical Highlight:** Uses `*args` and `**kwargs` for flexible function handling and `__doc__` inspection for auto-generated help menus.

### 2. 👥 OOP Employee Wrapper (Python)

An architectural demonstration of **Object-Oriented Programming** principles.

* **Hierarchy:** Features an `Employee` base class with specialized `Manager` and `Developer` subclasses.
* **Pillars of OOP:** Showcases inheritance, method overriding (polymorphism), and encapsulation.
* **Management System:** A complete wrapper for creating, viewing, and deleting employee records.

### 3. 📝 Personal Journal Manager (Python)

A robust utility focusing on **File I/O** and **System Interaction**.

* **Persistence:** Handles reading and writing to local `.txt` files with automated timestamping.
* **Error Handling:** Implements `try-except` blocks to manage `PermissionError` and `FileNotFoundError`.
* **Utility:** Includes keyword searching and secure "wipe all" functionality with confirmation prompts.

### 4. 📊 SalesCaptain: Data Analyzer (Python/Pandas)

An advanced data science tool for processing and visualizing sales data.

* **Libraries:** Built with `Pandas`, `NumPy`, `Matplotlib`, and `Seaborn`.
* **Data Repair:** Automated cleaning for duplicates and missing values (Mean/Mode imputation).
* **Analytics:** Generates pivot tables, grouped aggregations, and statistical distributions.
* **Visualization:** Exports Bar charts, Scatter plots, and Box plots automatically.

### 5. 🏛️ Academy & Corporate SQL Suites

Advanced relational database designs for complex data structures.

* **University Management:** Models `Students`, `Instructors`, and `Enrollments` with complex joins and running totals.
* **Data Transformer:** Demonstrates advanced SQL logic including `CASE` statements, Window Functions (`RANK`, `OVER`), and string transformations.
* **Insights:** Queries for tiered discounts, salary benchmarking, and academic status tracking.
